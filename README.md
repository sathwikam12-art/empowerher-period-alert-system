# EmpowerHer – Smart Period Alert System

EmpowerHer is a smart wearable system designed to help differently-abled girls track menstrual cycles using sensors and smart alerts.

## Problem
Blind, deaf, or mentally challenged girls often face difficulty tracking menstrual cycles, affecting hygiene and health.

## Solution
EmpowerHer uses sensors to detect menstruation and sends alerts through vibration, voice, or visual signals via a wearable band and mobile app.

## Features
- Sensor-based period detection
- Wearable alert band
- Mobile app integration
- Voice, vibration, and visual alerts
- Designed for differently-abled users

## Technologies
- ESP32 / nRF52832
- BLE communication
- Arduino / ESP-IDF
- Android mobile app
- Sensors for moisture detection
